# Social Images app
A social web app, built in Laravel, to share images and post comments/likes.

## Dependencies
PHP v8.2 or higher\
Database: MongoDB server or [MongoDB Atlas](https://www.mongodb.com/products/platform/atlas-database)\
Node.js v18.19.0 or higher
